# Botpress (botpress)

Botpress is an open-source conversational AI platform for building, deploying, and managing chatbots with a visual flow editor and cloud hosting. Botpress Cloud exposes a REST Cloud Management API plus a Chat API for sending messages to deployed bots. Self-hosted Botpress remains open-source.

## APIs
1. **Botpress Cloud Management API** — REST API at `https://api.botpress.cloud` for managing bots, integrations, conversations, users, and analytics.
2. **Botpress Chat API** — Webhook-based API at `https://chat.botpress.cloud/{webhookUrl}/messages` for sending/receiving chat messages.

## Pricing Snapshot
- Pay-as-you-go: $0/mo + AI Spend, $5/mo AI credit, 500 messages, 1 seat, 100MB vector
- Plus: $89/mo with increased usage limits
- Team: $495/mo + AI Spend, 3 seats, 2GB vector, RBAC, 50K messages
- Managed: subscription
- Enterprise: custom (typically ~$2,000/month on multi-year terms)
- Self-Hosted: free (you pay infra + LLM)
